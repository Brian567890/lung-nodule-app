## 系统简介

### 概述
肺结节随访管理APP是一款面向胸外科医生的专业医疗工具，基于《肺结节诊治中国专家共识（2024年版）》开发，用于肺结节患者的个性化随访管理和诊疗策略指导。

### 核心功能
- ✅ 患者档案管理（唯一编号、身份信息、危险因素）
- ✅ 肺结节详细录入（大小、密度、影像特征）
- ✅ 恶性概率自动计算（梅奥模型）
- ✅ 随访计划智能生成（2024版专家共识）
- ✅ 随访提醒与日历
- ✅ 数据统计分析
- ✅ 数据导入导出

### 技术特点
- **离线使用**：数据存储在本地，无需网络
- **隐私安全**：患者数据不上传云端
- **专业算法**：基于权威指南的随访策略
- **双语支持**：中英文界面一键切换

## 安装与启动

### 系统要求
- Android 6.0 (API 23) 或更高版本
- 存储空间：100MB以上

### 安装步骤

#### 方法一：直接安装APK
1. 下载 `app-release.apk` 文件
2. 在手机上允许"安装未知来源应用"
3. 点击APK文件完成安装

#### 方法二：Flutter开发环境运行
```bash
# 1. 进入项目目录
cd lung_nodule_app

# 2. 安装依赖
flutter pub get

# 3. 连接设备并运行
flutter run

# 4. 构建发布版APK
flutter build apk --release
```

## 功能模块

### 1. 首页

#### 统计概览
- **患者总数**：系统中登记的所有患者数量
- **结节总数**：所有患者的肺结节总数
- **随访中结节**：当前处于随访状态的结节数量
- **逾期随访**：超过下次随访日期的结节数量

#### 快速操作
- 添加患者
- 查看患者列表
- 打开随访日历

#### 待随访提醒
显示未来30天内需要随访的患者列表，按紧急程度排序：
- 🔴 红色：已逾期
- 🟠 橙色：7天内到期
- 🔵 蓝色：其他

---

### 2. 患者管理

#### 添加患者
**基本信息**
- 姓名（必填）
- 年龄（必填）
- 性别
- 手机号（可选）
- 身份证号（可选）

**危险因素**（用于高危人群判定）
- 吸烟史 + 包年数
- 肿瘤史（>5年）
- 肺癌家族史
- 慢阻肺
- 肺结核史
- 弥漫性肺纤维化
- 高危职业暴露（石棉、铍、铀、氡等）

> 💡 系统会根据2024版专家共识自动判定是否为"肺癌高危人群"

#### 患者列表
- 搜索：支持姓名、手机号、身份证号模糊搜索
- 筛选：高危人群会有特殊标识
- 排序：按建档时间倒序排列

#### 患者详情
三个Tab页面：
1. **患者信息**：基本信息 + 危险因素汇总
2. **结节列表**：该患者的所有肺结节
3. **随访历史**：该患者的所有随访记录

---

### 3. 肺结节管理

#### 添加结节

**基本参数**
- 发现日期
- 直径（mm）- 恶性概率计算关键参数
- 密度类型：
  - 实性结节
  - 纯磨玻璃结节（pGGN）
  - 混杂性结节（mGGN）

**位置信息**
- 肺叶：右上/右中/右下/左上/左下
- 肺段（可选）
- 具体位置描述

**影像特征**（梅奥模型参数）
- 毛刺征 ⚠️ 重要参数
- 分叶征
- 胸膜凹陷征
- 血管集束征
- 空泡征
- 空洞

**CT值**（可选）
- CT值最小/最大/平均（Hu）

> 💡 填写直径和影像特征后，系统会**实时计算恶性概率**

#### 恶性概率解读

| 概率范围 | 风险等级 | 处理建议 |
|---------|---------|---------|
| < 5% | 低风险 | CT随访观察 |
| 5-65% | 中风险 | CT随访或进一步检查（结节>8mm建议PET-CT或活检） |
| > 65% | 高风险 | 建议手术切除或病理诊断 |

---

### 4. 随访管理

#### 随访计划规则（2024版专家共识）

**实性结节**
| 结节大小 | 随访方案 |
|---------|---------|
| ≤4mm | 常规年度检查 |
| 4-6mm | 6-12个月 → 18-24个月 → 年度 |
| 6-8mm | 3-6个月 → 9-12个月 → 每6个月（2年） |
| >8mm | 根据恶性概率：PET-CT/活检/手术 |

**纯磨玻璃结节（pGGN）**
| 结节大小 | 随访方案 |
|---------|---------|
| ≤5mm | 6个月 → 年度 |
| 5-10mm | 3个月确认 → 6个月 → AI+MDT评估 |
| >10mm | 考虑活检/手术 |

**混杂性结节（mGGN）**
| 结节大小 | 随访方案 |
|---------|---------|
| ≤8mm | 3、6、12、24个月影像随访 |
| >8mm | 3个月复查 → 持续存在则积极处理 |

#### 添加随访记录
每次复查时记录：
- 检查日期
- 检查方法（CT类型）
- 当前直径（自动计算变化）
- 变化情况：
  - 结节增大
  - 新增实性成分
  - 实性成分增加
  - 形态变化
- 评估结论
- 医生建议

> 💡 保存随访记录后，系统会自动更新下次随访日期

#### 随访日历
- 月视图展示所有随访安排
- 标记有随访的日期
- 点击日期查看当天随访列表
- 逾期提醒

### 6. 数据管理

#### 导出数据
支持三种格式：

1. **JSON格式**（推荐备份）
   - 包含完整数据
   - 可用于数据恢复
   - 文件名：`lung_nodule_backup_YYYYMMDD_HHMM.json`

2. **患者列表CSV**
   - 可用Excel打开
   - 包含患者基本信息和危险因素
   - 文件名：`patients_YYYYMMDD_HHMM.csv`

3. **结节数据CSV**
   - 可用Excel打开
   - 包含结节详细信息和随访计划
   - 文件名：`nodules_YYYYMMDD_HHMM.csv`

#### 导入数据
- 支持导入JSON格式的备份文件
- 导入时会自动为新数据生成唯一ID
- 不会覆盖现有数据，而是合并添加

#### 清空数据
- 删除所有患者和结节数据
- ⚠️ 操作不可恢复，建议先导出备份

## 操作流程

### 典型工作流程

```
1. 添加患者
   └── 录入基本信息和危险因素
   
2. 添加结节
   └── 录入CT发现的结节参数
   └── 系统自动计算恶性概率和随访计划
   
3. 定期随访
   └── 按系统提醒进行复查
   └── 添加随访记录
   └── 系统更新下次随访日期
   
4. 数据分析
   └── 查看统计报表
   └── 导出数据用于科研或汇报
```

### 首次使用建议

1. **设置语言**：进入设置 → 选择中文或English
2. **添加测试患者**：创建1-2个测试患者熟悉流程
3. **添加测试结节**：尝试不同密度类型和大小的结节，观察随访计划差异
4. **查看统计**：添加几个患者后查看统计分析页面

## 数据管理

### 数据安全
- 所有数据存储在手机本地SQLite数据库
- 不连接任何网络服务
- 不涉及患者隐私数据传输

### 备份建议
- 定期导出JSON格式备份
- 重要数据变更后备份
- 导出文件可通过微信/邮件等方式保存到其他设备

### 迁移数据到新手机
1. 在原手机导出JSON备份
2. 将备份文件传输到新手机
3. 在新手机安装APP
4. 使用导入功能恢复数据

## 常见问题

### Q: 恶性概率计算准确吗？
A: 采用梅奥诊所模型（Mayo Clinic Model），该模型基于大量临床数据建立，在肺结节风险评估中被广泛使用。但任何模型都有局限性，建议结合临床判断和MDT讨论。

### Q: 随访计划符合最新指南吗？
A: 是的，随访规则基于《肺结节诊治中国专家共识（2024年版）》制定，这是目前国内最新的肺结节管理指南。

### Q: 数据会丢失吗？
A: 数据保存在本地，只要不清除APP数据或卸载APP，数据就会保留。建议定期导出备份以防意外。

### Q: 支持多设备同步吗？
A: 目前不支持云端同步。如需多设备使用，可通过导出/导入JSON文件手动同步数据。

### Q: 可以录入多发结节吗？
A: 可以。一个患者可以录入多个结节，每个结节独立管理，有各自的随访计划。

### Q: 随访提醒如何工作？
A: 打开APP时首页会显示待随访列表，建议医生每天查看。暂不支持系统通知栏推送（后续版本可能添加）。

### Q: 如何更新患者信息？
A: 患者详情页右上角有编辑按钮，可修改基本信息和危险因素。结节信息也可在结节详情中编辑。

## 技术参数

### 梅奥恶性概率模型公式
```
恶性概率 = e^x / (1 + e^x)

x = -6.8272 
  + (0.0391 × 年龄) 
  + (0.7917 × 吸烟史[0/1]) 
  + (1.3388 × 肿瘤史[0/1]) 
  + (0.1274 × 直径mm) 
  + (1.0407 × 毛刺征[0/1]) 
  + (0.7838 × 上叶位置[0/1])
```

### 风险分层标准
- 低风险：< 5%
- 中风险：5% - 65%
- 高风险：> 65%
